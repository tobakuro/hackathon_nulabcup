# GitQuiz Battle

**GitQuiz Battle** は、**自分の GitHub リポジトリから AI が自動生成したクイズ**で遊べる、プログラマー向けの対戦型クイズアプリです。
ヌーラボハッカソンで開発されたプロダクトです。

> 詳細: https://topaz.dev/projects/52f19d2cf881a25e101b

## 機能

### 対戦バトル
- マッチングで1対1のリアルタイム対戦
- **自分と相手のリポジトリ**から Gemini AI が問題を自動生成
- 1問 15 秒の制限時間・10ターン制
- 「ヌー」というゲーム内通貨をベットして勝敗を競う

### ソロプレイ
- 1人で自分のリポジトリのクイズに挑戦
- 難易度を選択して AI が 4 択問題を生成
- スコア履歴を記録・確認可能

### Code GeoGuessr
- リポジトリのランダムなコード 1 行を見て、ファイルと行番号を特定するクイズ
- ファイルツリーを探索しながら正確な場所を当てる
- 精度に応じてスコア計算

## 技術スタック

| 層 | 技術 |
|----|------|
| フロントエンド | Next.js 15 + React |
| バックエンド | Go 1.24（WebSocket によるリアルタイム通信） |
| 問題生成 | Gemini API（コードを解析して自動生成） |
| 認証 | GitHub OAuth（NextAuth.js） |
| DB | PostgreSQL（Supabase） |
| キャッシュ | Redis（Upstash） |
